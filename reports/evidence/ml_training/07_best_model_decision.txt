RubyBets - ML baseline 1X2
07 - Best model decision

Objective:
Compare several supervised classification models to predict the 1X2 football match result:
- HOME_WIN
- DRAW
- AWAY_WIN

Models compared:
1. DummyClassifier_most_frequent
2. LogisticRegression_balanced
3. RandomForest_balanced
4. XGBoost_classifier

Model comparison summary:
DummyClassifier_most_frequent:
- Accuracy: 0.4361
- F1 macro: 0.2024
- F1 weighted: 0.2648
- Limitation: predicts only HOME_WIN.

LogisticRegression_balanced:
- Accuracy: 0.4669
- F1 macro: 0.4266
- F1 weighted: 0.4525
- Strength: best global balance between the three classes.
- Limitation: DRAW remains difficult to predict.

RandomForest_balanced:
- Accuracy: 0.4417
- F1 macro: 0.4205
- F1 weighted: 0.4415
- Strength: better DRAW f1-score than Logistic Regression.
- Limitation: lower global performance.

XGBoost_classifier:
- Accuracy: 0.4872
- F1 macro: 0.3501
- F1 weighted: 0.4038
- Strength: best accuracy.
- Limitation: almost does not predict DRAW, with DRAW f1-score = 0.01.

Decision:
The selected model for this first 1X2 ML baseline is LogisticRegression_balanced.

Justification:
Logistic Regression is retained because it offers the best compromise between accuracy, F1 macro, F1 weighted, interpretability and stability.
Even though XGBoost has the best accuracy, it performs very poorly on DRAW predictions, which makes it less balanced and less defensible for RubyBets.
Random Forest predicts DRAW better, but its global scores are slightly lower than Logistic Regression.
DummyClassifier remains only a naive reference baseline.

Important limitation:
This model is a first experimental ML baseline.
It is not yet integrated into the RubyBets application.
It does not replace the current V1 scoring engine.
It must not be presented as a guaranteed sports prediction model.

Next recommended step:
Save the selected LogisticRegression_balanced model only if the current baseline is considered acceptable as a first experimental proof.
